How to make changes to and commit a file
Step 1: install git for windows GUI (first time only)
Step 2: Login with your gitHub credentials (first time only)
Step 3: Clone UTDSE4485 to your local machine
Step 4: Right click the repo and click "open in explorer" 
Step 5: Open the file you wish to make changes  
Step 6: Make your changes
Step 7: Save your changes (dont forget to change the file name!)
Step 8: double click the repo name in the Git for Windows client
Step 9: Add a description and commit
Step 10: click "sync"
